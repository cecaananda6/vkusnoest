# Кулинарный Портал "ВкусноЕст"

Современный интерактивный кулинарный сайт с рецептами, статьями и живым общением.

## Особенности

- **Умный поиск и фильтры**: Поиск по ингредиентам, фильтрация по времени приготовления, тегам (ПП, Вегетарианское) и категориям.
- **Система отзывов**: Пользователи могут оставлять оценки и комментарии к рецептам (сохраняются в браузере).
- **Личный кабинет**: Регистрация и вход (данные хранятся локально), возможность добавлять рецепты в избранное.
- **Форум**: Создание тем и обсуждение кулинарных вопросов с другими пользователями.
- **Полезные статьи**: Раздел с советами и лайфхаками для поваров.
- **SEO Оптимизация**: Сайт оптимизирован для поисковых систем и социальных сетей.
- **Адаптивный дизайн**: Отлично смотрится на компьютерах и мобильных устройствах.

## Как запустить

Этот сайт не требует установки дополнительных программ или серверов (Node.js не нужен).

1. Откройте папку проекта.
2. Найдите файл `index.html`.
3. Дважды кликните по нему, чтобы открыть в вашем браузере.

## Структура проекта

- `index.html` - Главная страница (Каталог рецептов).
- `articles.html` - Раздел со статьями.
- `forum.html` - Форум для общения.
- `css/styles.css` - Стили оформления.
- `js/` - Скрипты (app.js, auth.js, forum.js).
- `data/recipes.json` - База данных рецептов.

## Технологии

- HTML5, CSS3 (Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- LocalStorage (для хранения данных пользователей и отзывов)
- FontAwesome (Иконки)
- Google Fonts (Шрифты)
